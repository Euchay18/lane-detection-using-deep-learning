A deep learning model for lane detection for autonomous vehicles
Framework: Py Torch
GPU: A100 GPU
development environment: Google colab
link to dataset: https://www.kaggle.com/datasets/greatgamedota/culane/data 
used only a selected number of road images and its labels from the dataset due to inadequate computational resource
model achieved an accuracy of 75%
This project is about developing a lane detection model and since it is a computer vision project, deep learning models were the ideal choice.
Three deep learning models were explored; LaneNet, ENet and SegNet. SegNet was the best performing model and by default, the chosen model.
Only the python code for SegNet (Segmentation Network) is available on here.
